# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект использует [семантическое версионирование](https://semver.org/lang/ru/).

## [1.0.0] — 2026-05-XX

Первый релиз. Полнофункциональное desktop-приложение для расчётов ГСМ.

### Добавлено

- **Калькулятор** в двух режимах (Classic и Engineering) с двухстрочным дисплеем,
  превью выражения и историей вычислений.
- **Плавающие виджеты** для конвертации л↔кг с привязкой к плотности.
  Встроенные виджеты: АИ-92, ДТ-Л, ДТ-З, ТС-1, Масла, ТЖ, ОЖ.
- **Создание пользовательских виджетов** с фиксированной или переменной плотностью.
- **Три цветовые темы** Light / Dark / Blue с тёмной полосой заголовка через DWM.
- **Локализация** русский / английский с переключением на лету.
- **Сохранение и восстановление сессии** при перезапуске
  (дисплей, история, открытые виджеты с позициями).
- **Клавиатурные шорткаты** для всех операций.
- **xUnit тесты** на сервисы и MainViewModel (~80 тестов с Moq).
- **GitHub Actions CI** для build/test + автоматическая сборка релизов
  при пуше тега `v*`.

### Архитектура

- MVVM с разделением на Models / ViewModels / Views / Services.
- DI через Microsoft.Extensions.DependencyInjection.
- Хранение данных в `%AppData%\GsmCalculator\*.json`.
